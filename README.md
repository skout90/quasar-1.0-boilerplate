# Quasar App

## setup
Uninstall quasar-cli if you have it from <1.0 versions
  
> npm uninstall -g quasar-cli
  
Node.js >= 8.9.0 is required.
  
> npm install -g @quasar/cli

## library
- vuex
- moment
- lodash
- partial-js
- axios
